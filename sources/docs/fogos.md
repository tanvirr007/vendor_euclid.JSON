## Flashing Guide

### Clean Flash:
- Boot to Fastboot Mode
- fastboot boot [Boot](https://dl.twrp.me/fogos/twrp-3.7.1_12-0-fogos.img)
- Reboot to Recovery
- Flash the rom
- Format Data
- Reboot and Enjoy

### Dirty Flash:
- Boot to Fastboot Mode
- fastboot boot [Boot](https://dl.twrp.me/fogos/twrp-3.7.1_12-0-fogos.img)
- Reboot to Recovery
- Flash the rom
- Reboot and Enjoy
