## Flashing Guide

### Clean Flash
- Boot into recovery (OrangeFox/PBRP)
- Install the HyperOS 2.0 firmware for the respective region
- Install the ROM
- Re-flash recovery *(or enable Auto Reflash Recovery in TWRP)*
- Reboot to recovery
- **Format Data**  
  > Note: *Format Data is different from Wipe Data*
- Done — Reboot to system

---

### Dirty Flash
- Boot into recovery (OrangeFox/PBRP)
- Install the HyperOS 2.0 firmware for the respective region
- Install the ROM
- Re-flash recovery *(or enable Auto Reflash Recovery in TWRP)*
- Wipe **Dalvik / ART Cache**
- Done — Reboot to system
